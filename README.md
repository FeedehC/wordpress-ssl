# Docker-WordPress-SSL

WordPress + MySQL + phpMyAdmin + Nginx + SSL - CMS - Docker-compose

## Config

Copy "example.env" to ".env" and set environment variables.

## Start

docker-compose up -d

## Login

Login "localhost:80" or "example.com"

phpMyAdmin - "localhost:8080"
